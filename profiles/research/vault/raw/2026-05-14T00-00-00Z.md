## Raw Signal Notes
- **OWASP ASI (December 9, 2025)**: First formal OWASP Top 10 specifically for autonomous agents. 10 risks numbered ASI01–ASI10. Developed with 100+ experts. Key framing: "AI agents introduce new classes of vulnerabilities that extend beyond prompt-level attacks." Cascading failures documented as propagation through connected tools, memory, and other agents.
- **OWASP ASI case study — Amazon Q Developer (July 2025)**: Malicious GitHub PR submitted to ~1 million developers as v1.84.0. Hidden prompt instructed agent to wipe local files and delete AWS resources. Wiper failed only because prompt was malformed. Shows AI supply chain attack on development tooling — not just runtime agents.
- **OWASP ASI case study — EchoLeak (CVE-2025-32711)**: Zero-click Microsoft 365 Copilot injection. No user interaction required. Bypassed Microsoft's cross-prompt injection classifiers. Evaded link redaction. Routed data through trusted Microsoft Teams URLs. CVSS 9.3.

Key test: "If you cannot replay an agent decision from user request → model reasoning → retrieved context → tool call → final action, you do not have a production-grade agent system."
